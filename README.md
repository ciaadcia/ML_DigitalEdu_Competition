| Model   | Deskripsi |
|---------|-----------|
| Model 1 | Menggunakan **Logistic Regression** untuk memprediksi variabel `result`, yang merepresentasikan apakah seorang user akan melakukan pembelian course di platform **DigitalEdu**. Fitur yang digunakan mencakup karakteristik demografis, latar belakang pendidikan, dan aktivitas pengguna. |
| Model 2 | Memanfaatkan fitur **latar belakang pendidikan** untuk menganalisis korelasi antara tingkat pendidikan dan kecenderungan pembelian course. |
| Model 3 | Membandingkan beberapa pendekatan model dengan fokus fitur yang berbeda untuk memahami faktor yang paling berpengaruh terhadap keputusan pembelian course. |
