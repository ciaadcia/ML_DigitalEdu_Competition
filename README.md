MODEL 1 :

Model 1 menggunakan Logistic Regression untuk memprediksi variabel result, yang merepresentasikan apakah seorang user akan melakukan pembelian course di platform DigitalEdu. Fitur yang digunakan mencakup karakteristik demografis, latar belakang pendidikan, dan aktivitas pengguna.

MODEL 2 :

Model kedua memanfaatkan fitur latar belakang pendidikan untuk menganalisis korelasi antara tingkat pendidikan dan kecenderungan pembelian course.

MODEL 3:

Model 3 membandingkan beberapa pendekatan model dengan fokus fitur yang berbeda untuk memahami faktor apa yang paling berpengaruh terhadap keputusan pembelian course.
